					Quick Beam
					==========

Purpose
--------
*	Is a pre-processor to Latex Beamer
*	Allows simple beamer presentations to be created.
*	Creation and editing is quicker and easier than with beamer alone.

Its a Command Line thing
------------------------
*	This is currently a command line thing, with no GUI.

You need the following
----------------------
*	It is built using make.
*	It compiles using gcc, and you need CscNetLib. 
	*	https://github.com/drbraithw8/CscNetlib
*	You need latex beamer installed.

Building
--------
*	Build both the code and the documentation by typing "make".

Documentation
-------------
*	The documentation is provided in the form of a quickbeam file.
*	Typing "make" should generate a pdf file.
*	Also read the quickbeam file "documentation.qb".
